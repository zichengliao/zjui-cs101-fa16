{\bf Question 1.}     
Which of the following produces a valid plot given data in \texttt{x} and \texttt{y}?

   \begin{enumerate}[label=\Alph*]
   	\item
   	\begin{Verbatim}
	import matplotlib.pyplot as plt
	plt.plot( x,y )
   	\end{Verbatim}
   	\item
   	\begin{Verbatim}
	import matplotlib.plot as plot
	plot( x,y )
   	\end{Verbatim}
   	\item
   	\begin{Verbatim}
	import matplotlib.plot as plot
	plot( x,y )
   	\end{Verbatim}
   \end{enumerate}

\end{Verbatim}
