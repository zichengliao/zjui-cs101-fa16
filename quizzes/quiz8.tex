{\bf Question 1.}     \begin{lstlisting}[frame=single]
NameError:  name 'y' is not defined.
 \end{lstlisting}

Which of the following produces this error?

\begin{enumerate}[label=\Alph*]
 \item
  	\begin{lstlisting}[frame=single]
	x = 1
	y = x * 2
    \end{lstlisting}
 \item
	\begin{lstlisting}[frame=single]
	x = 0
	y += 1
    \end{lstlisting}
 \item
	\begin{lstlisting}[frame=single]
	x = 'ABCD'
	y = x[2]
	 \end{lstlisting}
\end{enumerate}
