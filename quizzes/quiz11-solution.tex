{\bf Question 1.}     

  \begin{Verbatim}
  x = 1
  y = true
  ~(x ~= y || y < x)
  \end{Verbatim}
  
  What is the result of this operation?
  
  \begin{enumerate}[label=\Alph*]
  	\item  \texttt{true}  \correctstar
  	\item  \texttt{false}
  	\item  \texttt{0}
  	\item  \texttt{1}  \correctstar
  \end{enumerate}
